#  E-Commerce Sales & Customer Behavior Analysis

## Project Overview
This project analyzes e-commerce sales and customer behavior data from a Turkish online retail platform.
The goal is to extract business insights using Python and SQL.

## Dataset
- Source: Kaggle – E-Commerce Customer Behavior and Sales Analysis (TR)
- Time Period: Jan 2023 – Mar 2024
- Records: 5,000 transactions

## Tools Used
- Python (pandas, matplotlib, seaborn)
- SQL (SQLite)
  

## Key Analyses
- Sales performance and revenue trends
- Product category and city-level analysis
- RFM customer segmentation
- Discount impact analysis
- Delivery time vs customer satisfaction

## Key Insights
- Returning customers generate a higher share of total revenue
- Electronics is the top revenue-generating category
- Faster delivery times are associated with higher customer ratings
- Medium discount levels balance volume and revenue best
